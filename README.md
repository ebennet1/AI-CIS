# AI Chief of Staff

AI Chief of Staff is a coordination assistant designed to help leaders and delivery teams manage meetings, decisions, action items, risks, dependencies, and executive communications across complex programs.

## Purpose

The project creates an AI-powered operating layer that turns fragmented information from meetings, notes, project updates, and approved documentation into structured outputs leaders can use immediately.

## Core Capabilities

- Meeting summaries with decisions, risks, blockers, and action items
- Weekly executive briefings
- Decision log generation and maintenance
- Risk, issue, and dependency identification
- Stakeholder communication drafts
- Action item follow-up support
- Source-grounded Q&A across approved project material

## MVP Scope

The MVP focuses on one pilot program or leadership forum and supports:

1. Meeting note summarization
2. Action item extraction
3. Decision tracking
4. Risk and blocker identification
5. Weekly executive status generation
6. Draft stakeholder communications

## Repository Structure

```text
docs/        Project charter, roadmap, operating model, and governance notes
prompts/     Reusable AI prompts for Chief of Staff workflows
templates/   Executive update, meeting summary, decision log, and tracker templates
examples/    Sample inputs and outputs
src/         Future application or automation code
```

## Initial Success Metrics

- Reduce time spent preparing weekly leadership updates by 30-50%
- Reduce manual meeting summary time by 50%+
- Improve action item completeness and owner clarity
- Improve decision traceability
- Improve consistency of executive communication
