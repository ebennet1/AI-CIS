# Meeting Summary Prompt

Summarize the meeting into the following sections:

1. Executive Summary
2. Key Decisions
3. Risks and Blockers
4. Action Items
5. Dependencies
6. Escalations Needed
7. Follow-Up Topics

For every action item include:
- Owner
- Due date if mentioned
- Priority
- Dependencies

For every risk include:
- Severity
- Impact
- Suggested mitigation
